# World Layoffs Data Visualization

## Project Overview

This section of the project focuses on building an interactive **Tableau Dashboard** to visualize global layoffs data and present key insights discovered during the exploratory data analysis phase.

The dashboard helps transform raw data into clear visual insights that allow users to understand trends in layoffs across different dimensions such as time, industry, company stage, and country.

## Key Visualizations

The dashboard includes several visualizations to explore layoffs trends:

### 1. Total Layoffs Over Time
A time series visualization showing layoffs trends by year.

### 2. Layoffs by Industry
Highlights which industries experienced the highest layoffs.

### 3. Layoffs by Country
Shows the geographical distribution of layoffs across countries.

### 4. Company Shutdown Analysis
Compares companies that completely shut down versus companies that continued operating.

### 5. Layoffs by Company Stage
Analyzes layoffs based on company funding stages (e.g., Startup, Post-IPO, etc.).
